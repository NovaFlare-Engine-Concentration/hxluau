
# hxluau
![](https://img.shields.io/github/repo-size/Ajwyunsx/hxluau) ![](https://badgen.net/github/open-issues/Ajwyunsx/hxluau) ![](https://badgen.net/badge/license/MIT/green)
Haxe/hxcpp @:native bindings for [Luau](https://luau.org/), the fast, small, safe, gradually typed embeddable scripting language derived from Lua.
> **⚠️ Warning**
> 
> Luau uses CMake build system, and static libraries may have binary incompatibility across different platforms and compilers. When using hxluau, ensure your development environment matches the precompiled library's build environment, or consider recompiling Luau from source for optimal compatibility.
### Installation
You can install it through `Haxelib`
```bash
haxelib install hxluau
```
Or through `Git`, if you want the latest updates
```bash
haxelib git hxluau https://github.com/Ajwyunsx/hxluau.git
```
